# 변경 이력 (Changelog)

## [1.0.27] - 2026-01-19

### 고쳐짐 (Fixed)
- **빌드 오류 수정**: 코드 리팩토링 과정에서 발생한 문법 오류(괄호 미일치)를 긴급 수정하여 정상 빌드되도록 조치했습니다.

## [1.0.26] - 2026-01-19

### 고쳐짐 (Fixed)
- **Web 무한 로딩 해결**: 스플래시 화면과 설정 확인 로직 간의 순환 참조(Infinite Loop)로 인해 앱이 계속 재시작되는 문제를 수정했습니다.

## [1.0.25] - 2026-01-19

### 디버그 (Debug)
- **Web 디버깅**: 스플래시 화면 멈춤 현상 정밀 분석을 위한 로그 추가 (DataSync, Hive)

## [1.0.24] - 2026-01-19

### 고쳐짐 (Fixed)
- **Web 스플래시 멈춤**: 데이터 확인 중 에러 발생 시(예: 네트워크 문제, CORS 등) 알림창 없이 멈추는 현상 수정. 이제 에러 발생 시 알림창을 통해 다운로드 화면으로 복구할 수 있습니다.

## [1.0.23] - 2026-01-19

### 고쳐짐 (Fixed)
- **팝업 깜빡임**: macOS 등 일부 환경에서 업데이트 팝업이 중복으로 떠서 깜빡이는 현상 수정
- **업데이트 실패**: 데이터 동기화 시 API 응답이 없으면 성공으로 오인하는 문제 해결 (실패 시 재시도 유도)
- **화면 전환**: 업데이트 버튼 클릭 시 다운로드 화면에서 바로 메인으로 튕기는 문제 수정

## [1.0.22] - 2025-12-29

### 개선됨 (Improved)
- **에러 핸들링**: 에러 처리 구조를 표준화(`Failure` 객체 기반)하여, 시스템 전반의 예외 처리를 일관성 있게 개선했습니다.
- **다국어 (L10n)**: 모든 에러 메시지와 안내 문구를 다국어 리소스(`arb`)와 연동하여, 사용자 언어 환경에 맞는 메시지를 제공합니다.
- **안정성**: 테스트 코드를 보강하고, 테스트 중 발견된 UI 관련 이슈(헤더 동작 등)를 수정했습니다.

## [1.0.21] - 2025-12-26
- UI 개선: 검색 탭을 헤더로 통합하여 스크롤 시 자동 숨김 처리
- UI 개선: 모바일 화면에서 검색 결과 리스트 상단 가림 현상 수정
- UI 개선: 데스크탑 레이아웃 탭바 중복 제거

## [1.0.20] - 2025-12-26

### 고쳐짐 (Fixed)
- **데이터 삭제**: 설정 화면의 '데이터 삭제' 기능이 작동하지 않던 버그를 수정했습니다. 이제 앱 데이터를 완전히 초기화하고 다운로드 화면으로 정상 이동합니다.
- **안정성**: 특정 상황에서 앱 실행 시 무한 로딩이 발생할 수 있는 잠재적 문제를 해결했습니다.

## [1.0.19] - 2025-12-26

### 새로운 기능 (New)
- **지능형 정제 규칙 동기화**: 앱 시작 시 원격 정제 규칙의 변경사항을 자동으로 감지하여 재정제합니다.
- **동기화 시간 표시**: 설정 화면에서 마지막 데이터 동기화 및 규칙 업데이트 시간을 확인할 수 있습니다.
- **안정성**: 데이터 다운로드 시 항상 최신 규칙을 먼저 적용하도록 동기화 순서를 보장합니다.

## [1.0.18] - 2025-12-26

## [1.0.3] - 2025-12-17

### 개선됨 (Improved)
- **레이아웃**: 메인 화면과 상세 화면의 너비 제한을 제거하여, 넓은 모니터에서도 화면을 꽉 채워 사용할 수 있도록 개선했습니다.
- **그리드 뷰**: 검색 결과 카드의 높이를 고정(Fixed Height)하여, 화면 너비가 넓어져도 카드가 불필요하게 길어지지 않도록 최적화했습니다.

## [1.0.2] - 2025-12-17

### 추가됨 (Added)
- **UI 디자인**: `Pretendard` 폰트를 적용하여 타이포그래피와 심미성을 개선했습니다.
- **데이터 정보**: 설정 화면에 저장된 데이터의 개수와 용량을 표시하는 기능을 추가했습니다.
- **가독성**: 검색 결과 카드에서 "신고번호"와 "주원료"의 시인성을 강화했습니다.

### 변경됨 (Changed)
- **UX**: 정보 파악이 쉽도록 검색 결과 카드의 레이아웃을 `[신고번호(상단)] -> [제품명] -> [내용]` 순서로 재배치했습니다.
- **상세 화면**: 라벨 텍스트의 명암비를 높이고 폰트 크기를 표준화했습니다.
- **검색**: 텍스트 잘림 방지를 위해 그리드 아이템의 비율을 조정했습니다.

### 수정됨 (Fixed)
- **버그**: 그리드 뷰에서 카드의 가로세로 비율을 조정하여 레이아웃 깨짐(RenderFlex overflow) 현상을 수정했습니다.
- **버그**: 리스트 뷰에서 잘못된 `Expanded` 사용으로 인한 무한 높이 오류(unbounded height)를 수정했습니다.
