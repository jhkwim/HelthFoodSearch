# 변경 이력 (Changelog)

## [1.0.7] - 2025-12-18

### 개선됨 (Improved)
- **품질 관리 (QA)**: 개발 표준 가이드(Rule #8)에 따라 비즈니스 로직과 UI에 대한 테스트 코드를 작성했습니다.
    - `IngredientSearchCubit` (단위 테스트)
    - `FoodItemHiveModel` (데이터 모델 테스트)
    - `SettingsScreen` (위젯/UI 테스트)

## [1.0.6] - 2025-12-17

### 고쳐짐 (Fixed)
- **배포 (CI/CD)**: iOS 빌드 워크플로우에서 발생하던 스크립트 문법 오류를 수정하여, 릴리즈 빌드가 정상적으로 수행되도록 조치했습니다.

## [1.0.5] - 2025-12-17

### 새로운 기능 (New)
- **다국어 지원 (L10n)**: 앱의 국제화를 위한 기초 작업을 완료했습니다.
    - 한국어(Korean) 리소스가 `app_ko.arb`로 분리되어 관리됩니다.
    - 설정, 메인 화면, 검색, 상세 정보 등 모든 UI 텍스트를 리소스 기반으로 변경하여 유지보수성을 높였습니다.
    - 추후 영어 등 다른 언어 추가가 용이해졌습니다.

## [1.0.4] - 2025-12-17

### 고쳐짐 (Fixed)
- **설정 화면**: 앱 버전이 `1.0.1`로 고정되어 보이던 문제를 수정했습니다. 이제 모든 플랫폼에서 현재 설치된 정확한 버전을 표시합니다.

## [1.0.3] - 2025-12-17

### 개선됨 (Improved)
- **레이아웃**: 메인 화면과 상세 화면의 너비 제한을 제거하여, 넓은 모니터에서도 화면을 꽉 채워 사용할 수 있도록 개선했습니다.
- **그리드 뷰**: 검색 결과 카드의 높이를 고정(Fixed Height)하여, 화면 너비가 넓어져도 카드가 불필요하게 길어지지 않도록 최적화했습니다.

## [1.0.2] - 2025-12-17

### 추가됨 (Added)
- **UI 디자인**: `Pretendard` 폰트를 적용하여 타이포그래피와 심미성을 개선했습니다.
- **데이터 정보**: 설정 화면에 저장된 데이터의 개수와 용량을 표시하는 기능을 추가했습니다.
- **가독성**: 검색 결과 카드에서 "신고번호"와 "주원료"의 시인성을 강화했습니다.

### 변경됨 (Changed)
- **UX**: 정보 파악이 쉽도록 검색 결과 카드의 레이아웃을 `[신고번호(상단)] -> [제품명] -> [내용]` 순서로 재배치했습니다.
- **상세 화면**: 라벨 텍스트의 명암비를 높이고 폰트 크기를 표준화했습니다.
- **검색**: 텍스트 잘림 방지를 위해 그리드 아이템의 비율을 조정했습니다.

### 수정됨 (Fixed)
- **버그**: 그리드 뷰에서 카드의 가로세로 비율을 조정하여 레이아웃 깨짐(RenderFlex overflow) 현상을 수정했습니다.
- **버그**: 리스트 뷰에서 잘못된 `Expanded` 사용으로 인한 무한 높이 오류(unbounded height)를 수정했습니다.
